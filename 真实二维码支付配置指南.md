## 🔧 配置步骤

### 第一步：获取支付宝配置信息

```bash
# 生成RSA密钥对
openssl genrsa -out rsa_private_key.pem 2048
openssl rsa -in rsa_private_key.pem -pubout -out rsa_public_key.pem

# 上传公钥到支付宝开放平台
# 获取支付宝公钥
```

### 第二步：配置application.yml

```yaml
payment:
  alipay:
    app-id: "202100000000000000"  # 你的支付宝AppID
    private-key: |
      -----BEGIN PRIVATE KEY-----
      你的RSA私钥内容
      -----END PRIVATE KEY-----
    public-key: |
      -----BEGIN PUBLIC KEY-----
      支付宝公钥内容
      -----END PUBLIC KEY-----
    gateway: "https://openapi.alipay.com/gateway.do"
    notify-url: "http://localhost:8080/api/third-party-payment/callback/alipay"
    return-url: "http://localhost:5173/payment/success"
  
  wechat:
    app-id: "wx0000000000000000"  # 你的微信AppID
    mch-id: "1900000000"          # 你的商户号
    api-key: "your-wechat-api-key" # 你的API密钥
    cert-path: "path/to/cert.p12"  # 证书路径
    notify-url: "http://localhost:8080/api/third-party-payment/callback/wechat"
    return-url: "http://localhost:5173/payment/success"
```

## 📱 二维码功能

### 生成的二维码特点：
- **真实支付链接** - 直接调用支付宝/微信API生成
- **自定义样式** - 包含支付平台标识
- **Base64格式** - 直接在网页显示
- **高清晰度** - 300x300像素，清晰可扫

### 二维码内容：
- **支付宝**: `https://qr.alipay.com/baxxxxxx`
- **微信**: `weixin://wxpay/bizpayurl?pr=xxxxxx`

## ⚠️ 注意事项

1. **安全配置**
   - 生产环境必须使用HTTPS
   - 妥善保管私钥
   - 验证所有回调签名

2. **网络配置**
   - 确保回调URL可访问
   - 配置防火墙规则
   - 使用内网穿透工具测试

3. **金额限制**
   - 测试环境限制金额
   - 生产环境设置合理限额
   - 防止重复支付

## 🛠 故障排除

### 常见问题：

1. **二维码生成失败**

2. **API调用失败**
